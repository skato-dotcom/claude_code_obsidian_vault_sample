# Invoca Terminology

> Source of truth for product vocabulary. Verify every outbound message against this list.
> Never use a synonym if the accepted term exists here.

| Accepted term | Do not use |
|---|---|
| Qualification pipeline | Lead scoring, lead pipeline, qualification flow |
| Data extraction | Data parsing, data scraping, extraction layer |
| Conversation intelligence | Call analytics, call insights |
| AI agent | Bot, virtual agent, chatbot |
| Caller | User, customer (unless context is clearly customer-facing) |
| Signal | Indicator, flag, marker |
| Transcript | Call log, call record |

> Add new terms as they are confirmed in Confluence or by the team.
